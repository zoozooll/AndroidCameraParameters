應用概覽

本頁面詳細介紹了應用程式的儀表板及其核心特性。

儀表板元件​

1. 導覽與選擇​

  • 導覽選單：快速存取設定、匯出/匯入、主題切換以及關於資訊。
  • 相機選擇：點擊相機名稱或 ID 徽章可在後置、前置和外置鏡頭之間切換。

2. 摘要卡​

摘要卡提供所選相機的高階快照：

  • 硬體層級：顯示 Camera2 API 支援層級（LEGACY、LIMITED、FULL、LEVEL_3）。
  • 感測器解析度：感測器的總百萬像素數。
  • 最大影片 FPS：影片錄製支援的最高幀率。

3. 核心特性網格​

快速概覽網格，顯示對關鍵專業特性的支援：

  • RAW 支援：擷取未壓縮感測器資料的能力。
  • 手動曝光與對焦：專業級的影像擷取控制。
  • 閃光燈功能：支援自動閃光和紅眼消除。
  • OIS (光學防手震)：硬體級別的震動減少。
  • HDR 與臉部偵測：智慧場景處理功能。

4. 參數分類​

詳細的相機特性按邏輯分類，便於探索：

  • 感測器：有效陣列大小、靈敏度、曝光時間範圍。
  • 鏡頭：光圈、焦距、對焦距離。
  • AE/AF/AWB：自動曝光、自動對焦和自動白平衡的詳細控制。
  • 輸出：支援的 JPEG、RAW 和 YUV 尺寸。
