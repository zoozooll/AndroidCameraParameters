# App Overview

This page provides a detailed breakdown of the application's dashboard and key features.

![App Overview](/img/designer/page1.png)

## Dashboard Components

### 1. Navigation & Selection
- **Navigation Menu**: Quick access to Settings, Export/Import, Theme toggles, and About info.
- **Camera Selection**: Click on the Camera Name or ID Badge to switch between Rear, Front, and External lenses.

### 2. Summary Card
The Summary Card provides a high-level snapshot of the selected camera:
- **Hardware Level**: Indicates the Camera2 API support level (LEGACY, LIMITED, FULL, LEVEL_3).
- **Sensor Resolution**: The total megapixel count of the sensor.
- **Max Video FPS**: The highest supported frame rate for video recording.

### 3. Key Features Grid
A quick-glance grid showing support for critical professional features:
- **RAW Support**: Ability to capture uncompressed sensor data.
- **Manual Exposure & Focus**: Pro-level control over image capture.
- **Flash Capabilities**: Support for Auto-Flash and Red-Eye reduction.
- **OIS (Optical Image Stabilization)**: Hardware-based shake reduction.
- **HDR & Face Detection**: Intelligent scene processing capabilities.

### 4. Parameter Categories
Detailed camera characteristics are grouped into logical categories for easy exploration:
- **Sensor**: Active array size, sensitivity, exposure time ranges.
- **Lens**: Aperture, focal length, focus distance.
- **AE/AF/AWB**: Detailed controls for Auto Exposure, Auto Focus, and Auto White Balance.
- **Output**: Supported JPEG, RAW, and YUV sizes.
