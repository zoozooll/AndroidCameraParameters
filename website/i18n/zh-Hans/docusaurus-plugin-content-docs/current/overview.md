# 应用概览

本页面详细介绍了应用程序的仪表板和关键功能。

![应用概览](/img/designer/page1.png)

## 仪表板组件

### 1. 导航与选择
- **导航菜单**：快速访问设置、导出/导入、主题切换和关于信息。
- **摄像头选择**：点击摄像头名称或 ID 徽章可在后置、前置和外置镜头之间切换。

### 2. 摘要卡
摘要卡提供所选摄像头的概览：
- **硬件级别**：指示 Camera2 API 支持级别（LEGACY, LIMITED, FULL, LEVEL_3）。
- **传感器分辨率**：传感器的总像素。
- **最大视频帧率**：支持的最高视频录制帧率。

### 3. 关键功能网格
快速浏览网格显示对关键专业功能的支持情况：
- **RAW 支持**：捕捉未经压缩的传感器数据的能力。
- **手动曝光和聚焦**：对图像捕捉进行专业级控制。
- **闪光灯功能**：支持自动闪光和红眼消除。
- **OIS (光学防抖)**：基于硬件的抖动消除。
- **HDR 和面部检测**：智能场景处理能力。

### 4. 参数类别
详细的摄像头特性按逻辑分类，方便探索：
- **传感器**：活动数组大小、灵敏度、曝光时间范围。
- **镜头**：光圈、焦距、聚焦距离。
- **AE/AF/AWB**：自动曝光、自动聚焦和自动白平衡的详细控制。
- **输出**：支持的 JPEG、RAW 和 YUV 尺寸。
