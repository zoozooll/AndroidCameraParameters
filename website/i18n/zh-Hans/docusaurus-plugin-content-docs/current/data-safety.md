# 数据安全指南

本文档概述了 Android Camera Parameters 应用程序的数据收集和隐私惯例。

## 概览

Android Camera Parameters 采用“隐私至上”的设计理念。作为一款诊断工具，它需要访问硬件信息才能运行，但它不会收集或传输个人数据。

## 权限

### 摄像头权限 (`android.permission.CAMERA`)
- **要求**：访问 `CameraManager` 并检索 `CameraCharacteristics` 所必需的。
- **用法**：应用仅读取硬件元数据。它**不会**在没有用户明确操作的情况下录制视频或拍摄照片。

## 数据收集

- **个人信息**：应用**不会收集**姓名、电子邮件地址、电话号码或任何其他个人标识符。
- **位置数据**：应用**不会访问**您的 GPS 或网络位置。
- **硬件元数据**：应用读取摄像头镜头的技术规格（分辨率、焦距、支持的模式）。除非您明确使用“导出 JSON”功能进行共享，否则这些数据将保留在您的设备上。

## 数据共享

该应用程序**不会**与第三方共享任何数据。核心应用中没有集成任何跟踪 SDK（如 Firebase Analytics 或 Facebook SDK）。

## 用户控制

- **JSON 导出**：用户可以选择复制或共享原始摄像头参数 JSON。这完全由用户发起。
- **权限**：您可以随时通过 Android 系统设置撤销摄像头权限，但如果没有该权限，应用将无法显示摄像头详细信息。
