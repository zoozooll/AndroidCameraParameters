# アプリの概要

このページでは、アプリケーションのダッシュボードと主な機能の詳細な内訳について説明します。

![アプリの概要](/img/designer/page1.png)

## ダッシュボードコンポーネント

### 1. ナビゲーションと選択
- **ナビゲーションメニュー**: 設定、エクスポート/インポート、テーマの切り替え、およびバージョン情報へのクイックアクセス。
- **カメラの選択**: カメラ名または ID バッジをクリックして、背面、前面、および外部レンズを切り替えます。

### 2. サマリーカード
サマリーカードは、選択したカメラのハイレベルなスナップショットを提供します。
- **ハードウェアレベル**: Camera2 API サポートレベル (LEGACY、LIMITED、FULL、LEVEL_3) を示します。
- **センサー解像度**: センサーの総画素数。
- **最大ビデオ FPS**: ビデオ録画でサポートされている最高フレームレート。

### 3. 主な機能グリッド
重要なプロフェッショナル機能のサポートを示すクイック一覧グリッド:
- **RAW サポート**: 非圧縮センサーデータをキャプチャする機能。
- **手動露出とフォーカス**: 画像キャプチャのプロレベルの制御。
- **フラッシュ機能**: オートフラッシュと赤目軽減のサポート。
- **OIS (光学手ブレ補正)**: ハードウェアベースの手ブレ補正。
- **HDR と顔検出**: インテリジェントなシーン処理機能。

### 4. パラメータカテゴリ
詳細なカメラ特性は、探索しやすいように論理的なカテゴリにグループ化されています。
- **センサー**: アクティブアレイサイズ、感度、露出時間の範囲。
- **レンズ**: 絞り、焦点距離、フォーカス距離。
- **AE/AF/AWB**: 自動露出、オートフォーカス、自動ホワイトバランスの詳細な制御。
- **出力**: サポートされている JPEG、RAW、および YUV サイズ。
