# Обзор приложения

На этой странице представлен подробный разбор панели мониторинга приложения и его ключевых функций.

![Обзор приложения](/img/camera_params_feature_graph.png)

## Компоненты панели мониторинга

### 1. Навигация и выбор
- **Меню навигации**: быстрый доступ к настройкам, экспорту/импорту, переключению тем и информации «О программе».
- **Выбор камеры**: нажмите на название камеры или значок идентификатора, чтобы переключиться между основной, фронтальной и внешними линзами.

### 2. Карточка сводки
Карточка сводки содержит высокоуровневый снимок выбранной камеры:
- **Уровень оборудования**: указывает уровень поддержки Camera2 API (LEGACY, LIMITED, FULL, LEVEL_3).
- **Разрешение сенсора**: общее количество мегапикселей сенсора.
- **Макс. FPS видео**: самая высокая поддерживаемая частота кадров для записи видео.

### 3. Сетка ключевых функций
Сетка быстрого просмотра, показывающая поддержку критически важных профессиональных функций:
- **Поддержка RAW**: возможность захвата несжатых данных сенсора.
- **Ручная экспозиция и фокус**: контроль профессионального уровня над захватом изображений.
- **Возможности вспышки**: поддержка авто-вспышки и подавления эффекта красных глаз.
- **OIS (Оптическая стабилизация изображения)**: аппаратное снижение дрожания.
- **HDR и обнаружение лиц**: интеллектуальные возможности обработки сцены.

### 4. Категории параметров
Подробные характеристики камеры сгруппированы в логические категории для удобства изучения:
- **Сенсор**: размер активной области, чувствительность, диапазоны времени экспозиции.
- **Объектив**: диафрагма, фокусное расстояние, дистанция фокусировки.
- **AE/AF/AWB**: подробные элементы управления автоэкспозицией, автофокусом и автоматическим балансом белого.
- **Выход**: поддерживаемые размеры JPEG, RAW и YUV.
