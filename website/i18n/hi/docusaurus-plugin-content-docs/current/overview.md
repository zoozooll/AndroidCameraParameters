# ऐप अवलोकन

यह पेज एप्लिकेशन के डैशबोर्ड और प्रमुख विशेषताओं का विस्तृत विवरण प्रदान करता है।

![ऐप अवलोकन](/img/designer/page1.png)

## डैशबोर्ड घटक

### 1. नेविगेशन और चयन
- **नेविगेशन मेनू**: सेटिंग्स, निर्यात/आयात, थीम टॉगल और 'के बारे में' जानकारी तक त्वरित पहुंच।
- **कैमरा चयन**: रियर, फ्रंट और बाहरी लेंस के बीच स्विच करने के लिए कैमरा नाम या आईडी बैज पर क्लिक करें।

### 2. सारांश कार्ड
सारांश कार्ड चयनित कैमरे का उच्च-स्तरीय स्नैपशॉट प्रदान करता है:
- **हार्डवेयर स्तर**: Camera2 API समर्थन स्तर (LEGACY, LIMITED, FULL, LEVEL_3) को दर्शाता है।
- **सेंसर रिज़ॉल्यूशन**: सेंसर की कुल मेगापिक्सेल संख्या।
- **अधिकतम वीडियो FPS**: वीडियो रिकॉर्डिंग के लिए समर्थित उच्चतम फ्रेम दर।

### 3. प्रमुख विशेषता ग्रिड
महत्वपूर्ण पेशेवर विशेषताओं के समर्थन को दिखाने वाला एक त्वरित-झलक ग्रिड:
- **RAW समर्थन**: असम्पीडित सेंसर डेटा कैप्चर करने की क्षमता।
- **मैनुअल एक्सपोज़र और फोकस**: इमेज कैप्चर पर प्रो-लेवल नियंत्रण।
- **फ़्लैश क्षमताएं**: ऑटो-फ़्लैश और रेड-आई रिडक्शन के लिए समर्थन।
- **OIS (ऑप्टिकल इमेज स्टेबिलाइजेशन)**: हार्डवेयर-आधारित शेक रिडक्शन।
- **HDR और फेस डिटेक्शन**: बुद्धिमान दृश्य प्रसंस्करण क्षमताएं।

### 4. पैरामीटर श्रेणियां
आसान अन्वेषण के लिए विस्तृत कैमरा विशेषताओं को तार्किक श्रेणियों में समूहीकृत किया गया है:
- **सेंसर**: सक्रिय सरणी आकार, संवेदनशीलता, एक्सपोज़र समय सीमा।
- **लेंस**: अपर्चर, फोकल लंबाई, फोकस दूरी।
- **AE/AF/AWB**: ऑटो एक्सपोज़र, ऑटो फोकस और ऑटो व्हाइट बैलेंस के लिए विस्तृत नियंत्रण।
- **आउटपुट**: समर्थित JPEG, RAW और YUV आकार।
