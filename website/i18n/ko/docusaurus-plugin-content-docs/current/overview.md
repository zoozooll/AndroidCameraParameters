# 앱 개요

이 페이지는 애플리케이션의 대시보드와 주요 기능에 대한 자세한 분석을 제공합니다.

![앱 개요](/img/designer/page1.png)

## 대시보드 구성 요소

### 1. 탐색 및 선택
- **탐색 메뉴**: 설정, 내보내기/가져오기, 테마 전환 및 정보 페이지로 빠르게 이동합니다.
- **카메라 선택**: 카메라 이름이나 ID 배지를 클릭하여 후면, 전면 및 외부 렌즈 간에 전환할 수 있습니다.

### 2. 요약 카드
요약 카드는 선택한 카메라의 상위 수준 스냅샷을 제공합니다.
- **하드웨어 레벨**: Camera2 API 지원 레벨(LEGACY, LIMITED, FULL, LEVEL_3)을 나타냅니다.
- **센서 해상도**: 센서의 총 메가픽셀 수입니다.
- **최대 비디오 FPS**: 비디오 녹화에서 지원되는 가장 높은 프레임 속도입니다.

### 3. 주요 기능 그리드
중요한 전문 기능 지원 여부를 보여주는 퀵 글랜스 그리드:
- **RAW 지원**: 압축되지 않은 센서 데이터를 캡처하는 기능입니다.
- **수동 노출 및 초점**: 이미지 캡처에 대한 전문가 수준의 제어 기능을 제공합니다.
- **플래시 기능**: 자동 플래시 및 적목 현상 방지 기능을 지원합니다.
- **OIS (광학 이미지 흔들림 보정)**: 하드웨어 기반의 흔들림 감소 기능입니다.
- **HDR 및 얼굴 인식**: 지능형 장면 처리 기능을 제공합니다.

### 4. 파라미터 카테고리
상세한 카메라 특성은 탐색하기 쉽도록 논리적 카테고리로 그룹화되어 있습니다.
- **센서**: 활성 어레이 크기, 감도, 노출 시간 범위 등입니다.
- **렌즈**: 조리개, 초점 거리, 초점 거리 등입니다.
- **AE/AF/AWB**: 자동 노출, 자동 초점 및 자동 화이트 밸런스에 대한 세부 제어 기능입니다.
- **출력**: 지원되는 JPEG, RAW 및 YUV 크기입니다.
