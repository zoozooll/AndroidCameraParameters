## Welcome to Android Camera Parameters

Android Camera Parameters is a powerful diagnostic tool for developers and enthusiasts to explore the deep technical capabilities of their device's cameras.

![App Overview](docs/page1.png)

### Key Features

*   **Detailed Diagnostics**: Inspect `CameraCharacteristics` for all lenses.
*   **Hardware Level Detection**: Instantly see if your device supports `FULL` or `LEVEL_3` features.
*   **Real-time Feature Tracking**: Check for OIS, RAW support, Manual Focus, and more.
*   **Categorized View**: Explore hundreds of parameters organized by Sensor, Lens, and Processing categories.
*   **Raw Data**: View the complete camera profile as a structured JSON.

For more details, see the [Full Overview](docs/Overview.md).
